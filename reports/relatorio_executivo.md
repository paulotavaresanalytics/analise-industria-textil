# Relatório Executivo — Indústria Têxtil Fictícia

Empresa fictícia: **Fio & Forma Confecções Ltda.**  
Porte: **médio**, aproximadamente **200 colaboradores**  
Período analisado: **jan/2025 a dez/2025**

## Objetivo do projeto
Este projeto simula uma análise de dados de uma indústria têxtil com foco em cinco métricas de negócio:
1. Giro de estoque por coleção
2. Previsão de demanda por estação
3. Produtos mais vendidos
4. Devoluções por produto
5. Margem por peça

## Principais insights
- **Maior giro de estoque:** Linha Contínua (14.65)
- **Produto mais vendido:** Camiseta Básica Premium (2672 unidades)
- **Maior taxa de devolução:** Calça Sarja Slim (4.98%)
- **Maior margem unitária:** Casaco Lã Blend (R$ 127.9)
- **Estação com maior demanda histórica:** Primavera

## Observações metodológicas
- Os dados são **100% fictícios**, criados para fins de portfólio.
- A previsão de demanda usa uma lógica simples baseada em sazonalidade histórica e crescimento planejado de 8%.
- O cálculo de margem por peça considera: **preço de venda - custo unitário**.
- O giro de estoque foi estimado por: **quantidade vendida / estoque médio**.
